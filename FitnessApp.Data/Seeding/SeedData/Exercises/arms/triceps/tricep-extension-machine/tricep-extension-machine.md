## Form Cues
1. Adjust seat and grip, elbows aligned with pivot.
2. Push handles down or back until arms extended.
3. Return slowly to start.

## Common Mistakes
- Elbows flaring  
- Using momentum  

## Variations
**Easier:**  
- Lighter weight  
- Single-arm mode  

**Harder:**  
- Slow eccentric  
- Pause at contraction  

## Programming Options
- 3–4 sets × 10–12 reps  

## Benefits
- Triceps isolation  
- Safe for joints  

## Tips for Success
- Keep elbows tucked  
- Smooth motion
