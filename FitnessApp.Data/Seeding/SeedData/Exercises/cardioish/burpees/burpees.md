## Form Cues
1. Start standing.  
2. Drop into squat, hands on floor.  
3. Kick feet back into plank.  
4. Optional push-up.  
5. Jump feet back to squat.  
6. Explode up, arms overhead.  
7. Land softly and repeat.

## Common Mistakes
- Sagging hips  
- Skipping push-up  
- Landing hard  
- Going too fast  
- Poor plank form  

## Variations
**Easier:**  
- Step back instead of jump  
- No push-up  
- Hands on bench  

**Harder:**  
- Extra push-up  
- Burpee box jumps  
- Burpee pull-ups  
- Hold dumbbells  

## Programming Options
- HIIT: 30s work / 30s rest × 10–20 rounds  
- EMOM: 10–15 burpees at start of each minute  
- For time: 50–100 burpees as fast as possible  
- Circuit: include as one station  

## Benefits
- Full-body conditioning  
- Cardiovascular endurance  
- Burns calories  
- Builds mental toughness  

## Tips for Success
- Start manageable pace  
- Focus on form  
- Breathe rhythmically  
- Land softly
