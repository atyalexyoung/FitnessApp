## Form Cues
1. Stand feet shoulder-width, hold medicine ball overhead.  
2. Chop ball vertically down to one side, pivoting feet slightly.  
3. Return to start, repeat both sides.  

## Common Mistakes
- Rounding back  
- Using arms only  
- Poor pivoting  

## Variations
**Easier:**  
- Lighter ball  
- Slower pace  

**Harder:**  
- Heavier ball  
- Faster intervals  
- Include squat before chop  

## Programming Options
- 3–5 sets × 10–15 reps per side  
- Include in core/HIIT circuits  

## Benefits
- Core and oblique strength  
- Rotational power  
- Cardiovascular conditioning  

## Tips for Success
- Engage core  
- Controlled movement  
- Slight pivot for hips
