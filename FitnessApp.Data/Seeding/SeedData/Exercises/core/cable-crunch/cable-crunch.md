## Form Cues
1. Kneel facing cable machine with rope attachment.  
2. Hold rope by shoulders, elbows bent.  
3. Crunch down, contracting abs, then return slowly.  

## Common Mistakes
- Using momentum  
- Leaning forward  
- Arching lower back  

## Variations
**Easier:**  
- Lighter weight  
- Partial range of motion  

**Harder:**  
- Heavier weight  
- Slow negatives  
- Pause at contraction  

## Programming Options
- 3–5 sets × 10–15 reps  
- Include in core or ab circuits  

## Benefits
- Rectus abdominis strengthening  
- Weighted core training  
- Improves posture and control  

## Tips for Success
- Control movement  
- Engage abs fully  
- Avoid jerking
