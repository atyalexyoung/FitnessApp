## Form Cues
1. Hold end of landmine with both hands at chest.  
2. Perform squat, keeping core tight and back neutral.  
3. Return to standing under control.  

## Common Mistakes
- Leaning forward  
- Rounded back  
- Using momentum  

## Variations
**Easier:**  
- Lighter weight  
- Partial squat  

**Harder:**  
- Heavier bar  
- Full depth  
- Slow negative phase  

## Programming Options
- 3–5 sets × 8–12 reps  
- Include in leg or core circuits  

## Benefits
- Core stability  
- Leg and glute strengthening  
- Anti-rotation activation  

## Tips for Success
- Engage core  
- Keep back neutral  
- Move controlled
