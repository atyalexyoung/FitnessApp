## Form Cues
1. Position feet in GHR machine, knees on pad.
2. Lower torso slowly toward floor using hamstrings.
3. Pull back up to start using hamstrings and glutes.

## Common Mistakes
- Using momentum  
- Partial range  

## Variations
**Easier:**  
- Bodyweight only  
- Assistance from hands  

**Harder:**  
- Pause at bottom  
- Slow eccentric  

## Programming Options
- 3–4 sets × 8–12 reps  

## Benefits
- Hamstring and glute strength  
- Posterior chain development  
- Knee stability  

## Tips for Success
- Controlled motion  
- Full range
